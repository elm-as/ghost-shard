## 🎯 VISION
**Étude sur le stockage stéganographique décentralisé de données.**

Un prototype explorant l'obfuscation de données au sein d'images profil publiques. Ghost Shard fragmente les fichiers en pixels de bruit stéganographique, testant la viabilité d'une distribution sans serveur central.
